# React Wordle

这是广受欢迎的单词猜谜游戏 Wordle 的克隆版，使用 React、TypeScript 和 Tailwind CSS 构建。

[**在线体验 Demo！**](https://wordle-en.luomor.com/)

[English](README.md) | [简体中文](README_CN.md)

## 构建和运行

### 本地运行：

克隆仓库并执行以下命令：

```bash
$> cd react-wordle
$> npm install
$> npm run start
```

### Docker 容器：

#### 开发环境

```bash
$> docker build -t game:dev .
$> docker run -d -p 3000:3000 game:dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000)。

#### 生产环境

```bash
$> docker build --target=prod -t game:prod .
$> docker run -d -p 80:80 game:prod
```

在浏览器中打开 [http://localhost](http://localhost)。

## 常见问题

### 如何修改猜测单词的长度？

- 在 [src/constants/settings.ts](src/constants/settings.ts) 中更新 `MAX_WORD_LENGTH` 变量
- 在 [src/constants/wordlist.ts](src/constants/wordlist.ts) 中更新 `WORDS` 数组，仅包含新长度的单词
- 在 [src/constants/validGuesses.ts](src/constants/validGuesses.ts) 中更新 `VALID_GUESSES` 数组

### 如何创建其他语言版本？

- 在 [.env](.env) 中：
  - 更新标题和描述
  - 将 `REACT_APP_LOCALE_STRING` 设置为目标语言区域
- 在 [public/index.html](public/index.html) 中：
  - 更新 "You need to enable JavaScript" 提示
  - 更新 HTML 标签的 lang 属性
  - 如果是从右到左书写的语言，在 HTML 标签中添加 `dir="rtl"`
- 更新 [public/manifest.json](public/manifest.json) 中的 name 和 short_name
- 更新 [src/constants/strings.ts](src/constants/strings.ts) 中的所有字符串
- 将该语言的所有五字母单词添加到 [src/constants/validGuesses.ts](src/constants/validGuesses.ts)
- 将该语言的目标词列表添加到 [src/constants/wordlist.ts](src/constants/wordlist.ts)
- 更新"设置"弹窗 [src/components/modals/SettingsModal.tsx](src/components/modals/SettingsModal.tsx)
- 更新"说明"弹窗 [src/components/modals/InfoModal.tsx](src/components/modals/InfoModal.tsx)
- 如果该语言包含英语中没有的字母，更新键盘 [src/components/keyboard/Keyboard.tsx](src/components/keyboard/Keyboard.tsx)
- 如果是从右到左书写的语言，在 `generateEmojiGrid` 函数（[src/lib/share.ts](src/lib/share.ts)）的返回语句前添加 `‮`（Unicode 从右到左覆盖字符）

### 如何添加使用统计追踪？

本项目支持 Google Analytics 或 [Plausible Analytics](https://plausible.io)，默认关闭。

启用 Google Analytics：

- 创建 Google Analytics 4 媒体资源并获取测量 ID（格式为 `G-XXXXXXXXXX`）
- 在 [.env](.env) 中添加 `REACT_APP_GOOGLE_MEASUREMENT_ID=G-XXXXXXXXXX`

注意：部分地区可能需要在获得用户同意后才能启用追踪，这由各下游项目自行实现。

启用 Plausible Analytics：

- 在 Plausible Analytics 中创建新网站，获取域名，例如 `example.app`
- 在 [.env](.env) 中添加 `REACT_APP_PLAUSIBLE_DOMAIN=example.app`
